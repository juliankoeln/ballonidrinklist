⚙️ Firebase einrichten (einmalig)

1. Gehe zu console.firebase.google.com → Neues Projekt
2. Realtime Database aktivieren (Region: europe-west1)
3. Regeln auf "read": true, "write": true setzen
4. Projekteinstellungen → Web-App → Config kopieren
5. Diese HTML-Datei öffnen → oben die Config eintragen → fertig!

Danach diese Datei auf Google Drive, OneDrive oder per Link teilen – alle sehen denselben Bestand in Echtzeit.
